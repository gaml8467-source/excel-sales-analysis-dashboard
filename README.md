# 📊 Excel Sales Dashboard

An interactive Excel Sales Dashboard built using SQL Server, Power Query, Pivot Tables, Pivot Charts, Power Pivot, and Slicers.

The project demonstrates the complete data analysis workflow, from extracting data from a SQL Server database to transforming it and building an interactive dashboard in Microsoft Excel.

---

## 📌 Project Overview

This dashboard provides insights into sales performance by analyzing products, categories, quantities, and sales trends.

The data was extracted from a local SQL Server database, transformed using Power Query, and visualized through Pivot Tables and Pivot Charts.

---

## 🚀 Technologies Used

- Microsoft Excel
- SQL Server
- Power Query
- Pivot Tables
- Pivot Charts
- Power Pivot
- Slicers

---

## 🛠 Data Preparation (ETL)

The data preparation process included:

- Connecting to SQL Server
- Selecting the required tables
- Merging multiple tables using Power Query
- Removing unnecessary columns
- Renaming columns
- Changing data types
- Removing null values
- Creating a Date column
- Loading the cleaned dataset into Excel

---

## 🗂 Database Tables

The dashboard was built using the following tables:

- FactSales
- Product
- ProductSubcategory
- ProductCategory
- Date
- Territory

---

## 📈 Dashboard Features

### KPI Cards

### Interactive Charts

- 📈 Sales Trend

- 🏆 Top 10 Products

The dashboard includes slicers to filter data dynamically, making it easier to analyze sales performance from different perspectives.
